# site-internet-type-blog
portfolio


Ce projet est une implémentation complète du site web type blog en HTML et CSS.
Il a été réalisé dans un but pédagogique afin de mettre en pratique les bases d'un site web.

Répartition du projet Ananta

FERTATE Walid:  
Accueil :  
- Nom du jeu + logo  
- Image ou bande-annonce du jeu (visuel accrocheur)  
- Phrase d’accroche / slogan (« Entrez dans l’univers de… »)  
- Bouton “Jouer maintenant” ou “Télécharger”  
- Brève présentation (2-3 lignes sur le concept du jeu)  
- Liens vers les autres pages (ex : « En savoir plus », « L’équipe », « Contact »)

LIN Jacques:  
A propos du jeu:  
- Gameplay : comment on joue, quels sont les objectifs  
- Caractéristiques principales :  
- Mode solo / multi  
- Personnages, armes, niveaux  
- Plateformes disponibles (PC, console, mobile…)  
- Captures d’écran  

MOYSAN Jordan:  
Contact :  
- Présentation de l’équipe (développeurs, artistes, musiciens…)  
- Liens vers les réseaux sociaux (Discord, Twitter, Instagram…)  
- Formulaire de contact ou adresse mail
